Problem

You have gathered $$$\mathbf{N}$$$ bags of candy and you want to distribute the candy amongst $$$\mathbf{M}$$$ kids. The $$$i$$$-th bag contains $$$\mathbf{C_i}$$$ pieces of candy. You want to make sure that every kid get the same amount of candy and that the number of pieces of candy they receive is the greatest possible. You can open each bag and mix all pieces of candy before distributing them to the kids.

How many pieces of candy will remain after you share the candy amongst kids, based on the rules described above?

Input

The first line of the input gives the number of test cases, $$$\mathbf{T}$$$. $$$\mathbf{T}$$$ test cases follow.

Each test case consists of two lines. The first line of each test case contains two integers: integer $$$\mathbf{N}$$$, the number of candy bags, and $$$\mathbf{M}$$$, the number of kids.

The next line contains $$$\mathbf{N}$$$ non-negative integers $$$\mathbf{C_1},\mathbf{C_2},\dots,\mathbf{C_N}$$$ representing array $$$\mathbf{C}$$$, where the $$$i$$$-th integer represents the number of candies in the $$$i$$$-th bag.

Output

For each test case, output one line containing Case #x: y, where x is the test case number (starting from $$$1$$$) and y is the number of candies that will remain if you divide candies between kids according to the rules described above.

Limits

Time limit: 60 seconds.
Memory limit: 1 GB.

Test Set 1

$$$1 \le \mathbf{T} \le 100$$$.
$$$1 \le \mathbf{N} \le 10^5$$$.
$$$1 \le \mathbf{M} \le 10^4$$$.
$$$0 \le \mathbf{C_i} \le 1000$$$, for all $$$i$$$ from $$$1$$$ to $$$\mathbf{N}$$$.

Sample Output

Case #1: 1
Case #2: 5

In Sample Case #1, we have $$$\mathbf{N}=7$$$ bags of candy. In total we have $$$1 + 2 + 3 + 4 + 5 + 6 + 7 = 28$$$ candies that we want to divide between $$$\mathbf{M} = 3$$$ kids. Every kid can get $$$9$$$ pieces of candy, so $$$28 - 3 \times 9 = 1$$$ pieces of candy will remain.

In Sample Case #2, we have $$$\mathbf{N}=5$$$ bags of candy. In total we have $$$7 + 7 + 7 + 7 + 7 = 35$$$ candies that we want to divide between $$$\mathbf{M} = 10$$$ kids. Every kid can get $$$3$$$ pieces of candy, so $$$35 - 10 \times 3 = 5$$$ pieces of candy will remain.
