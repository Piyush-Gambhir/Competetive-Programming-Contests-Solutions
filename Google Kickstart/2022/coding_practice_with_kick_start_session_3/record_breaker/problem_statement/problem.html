Record Breaker

Problem

Download the Starter Code!

Isyana is given the number of visitors at her local theme park on $$$\mathbf{N}$$$ consecutive days. The number of visitors on the $$$i$$$-th day is $$$\mathbf{V_i}$$$. A day is record breaking if it satisfies both of the following conditions:

  • Either it is the first day, or the number of visitors on the day is strictly larger than the number of visitors on each of the previous days.
  • Either it is the last day, or the number of visitors on the day is strictly larger than the number of visitors on the following day.
Note that the very first day could be a record breaking day!

Please help Isyana find out the number of record breaking days.

Input

The first line of the input gives the number of test cases, $$$\mathbf{T}$$$. $$$\mathbf{T}$$$ test cases follow. Each test case begins with a line containing the integer $$$\mathbf{N}$$$. The second line contains $$$\mathbf{N}$$$ integers. The $$$i$$$-th integer is $$$\mathbf{V_i}$$$ and represents the number of visitors on the $$$i$$$-th day.

Output

For each test case, output one line containing Case #$$$x$$$: $$$y$$$, where $$$x$$$ is the test case number (starting from $$$1$$$) and $$$y$$$ is the number of record breaking days.

Limits

Time limit: 20 seconds.
Memory limit: 1 GB.
$$$1 \le \mathbf{T} \le 100$$$.
$$$0 \le \mathbf{V_i} \le 2 \times 10^5$$$, for all $$$i$$$.

Test Set 1

$$$1 \le \mathbf{N} \le 1000$$$.

Test Set 2

$$$1 \le \mathbf{N} \le 2 \times 10^5$$$, for at most 10 test cases.
For the remaining cases, $$$1 \le \mathbf{N} \le 1000$$$.

Sample Input
save_alt
content_copy Copied!

Sample Output
save_alt

Case #1: 2
Case #2: 1
Case #3: 3
Case #4: 0

In Sample Case #1, the underlined numbers in the following represent the record breaking days: $$$1\,\mathbf{\underline{2}}\,0\,\mathbf{\underline{7}}\,2\,0\,2\,0$$$.

In Sample Case #2, only the last day is a record breaking day: $$$4\,8\,15\,16\,23\,\mathbf{\underline{42}}$$$.

In Sample Case #3, the first, the third, and the sixth days are record breaking days: $$$\mathbf{\underline{3}}\,1\,\mathbf{\underline{4}}\,1\,5\,\mathbf{\underline{9}}\,2\,6\,5$$$.

In Sample Case #4, there is no record breaking day: $$$9\,9\,9\,9\,9\,9$$$.
